Prediction and Optimization of 3D Print Time

A Senior Project by James Batchelder

Advisors: Richard Lethin and Steven Zucker

Abstract: The time and volume of material used to print a 3D object are dependent on rotation. Normally, print time is computed when the object is prepared for printing in a computationally lengthy process known as slicing. In this project, the print time is predicted 3 orders of magnitude faster, with some loss in accuracy, by using machine learning methods rather than traditional slicing algorithms. This is accomplished with a nonlinear feature extractor combined with an ordinary least squares model. This allows a user to quickly check several promising orientations of an object to reduce print time. In the over $2 billion 3D printing industry there is still no widely accepted, generalized algorithm to find the optimal rotation for an object. To solve this problem, a gradient descent algorithm, based on the predictor, is used to find possible optimal orientations.
